# QuanLyBanHang - Build 1.0

Đây là Build 1.0 của dự án QuanLyBanHang, được sử dụng cho kiểm thử thủ công trong học phần CT243H - Software Quality Assurance and Testing.

Tài liệu hướng dẫn và template báo cáo nằm trong thư mục [.docs](.docs).

## Tech Stack

- Ngôn ngữ: C#
- Nền tảng: .NET Framework 4.5
- Giao diện: Windows Forms
- Cơ sở dữ liệu: Microsoft SQL Server 2014 trở lên (khuyến nghị Express/Developer)
- Công cụ build: MSBuild

## Mục đích sử dụng

- Phục vụ cho việc thực hiện test thủ công.
- Làm cơ sở ghi nhận kết quả kiểm thử, lỗi phát hiện và nhận xét đánh giá.
- Tham chiếu đặc tả yêu cầu và tài liệu liên quan trong thư mục .docs.
